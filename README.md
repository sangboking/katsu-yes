# Katsu-Yes: 돈까스 맛집 지도

## 📜 프로젝트 소개

`Katsu-Yes`는 Naver Map API를 기반으로 한 돈까스 맛집 정보 제공 애플리케이션입니다. 사용자는 지도에서 돈까스 가게의 위치를 확인하고, 상세 정보를 볼 수 있습니다.

**주요 기능:**

- Naver Map API를 연동한 지도 기반 맛집 위치 표시
- 돈까스 가게 상세 정보 제공 (주소, 영업시간, 메뉴, 리뷰 등)
- 구글 OAuth2.0을 이용한 로그인 기능

**배포 주소:** [https://katsu-yes.vercel.app/](https://katsu-yes.vercel.app/)

## 🛠️ 기술 스택

- **Frontend:** Next.js(v16), TypeScript, Tailwind CSS
- **State Management:** Zustand, TanStack Query (React Query)
- **Backend & DB:** Supabase
- **Deployment:** Vercel

## 💾 데이터베이스

현재 돈까스 맛집 가게 데이터베이스는 지속적 등록중에 있습니다.
